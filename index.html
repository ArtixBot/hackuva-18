Wisper

A personal security program that alerts you to sensitive personal data input.

How it Works

Wisper is a free, open-source technology that keeps track of your keyboard input and cross-checks it with a provided list of "personal information fields." In the event that you type out some phrase or number which corresponds with one of these fields, Wisper will log this information for future reference, allowing to see where, what, and when did you type out this information.

In the event of a data breach, you can use Wisper to quickly check if you've input personal information at the compromised sites, allowing you to take quick action to protect your identity.

Wisper requires no online connection and encrypts all personal information fields, so your data is kept safe from attacks.
